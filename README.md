# Proposta de nova pàgina corporativa per a FoodLogistic S.A.

## Presentació

Aquest document recull la proposta de prototip web creada per a **FoodLogistic S.A.** amb l’objectiu de renovar la seva presència digital i oferir una imatge més moderna, clara i professional a Internet.

La proposta s’ha desenvolupat com una **web corporativa estàtica**, visualment actualitzada, responsive i preparada per ser publicada sota una URL pública. També s’hi ha integrat una eina d’analítica per mesurar l’activitat de la pàgina i obtenir dades bàsiques de visites.

## Objectiu del prototip

La finalitat principal d’aquesta proposta és mostrar com podria ser la nova identitat digital de FoodLogistic S.A., millorant tant l’aspecte visual com l’estructura de la informació.

Amb aquest prototip es busca:

- transmetre una imatge més professional i actual
- presentar els serveis de manera clara
- facilitar el contacte amb possibles clients
- disposar d’una base web senzilla, accessible i fàcil d’ampliar
- incorporar una primera capa de control analític de visites

## Estructura del lloc web

La proposta inclou una pàgina corporativa amb les seccions principals següents:

### Inici
Espai de presentació principal amb missatge corporatiu, proposta de valor i primera impressió visual de la marca.

### Serveis
Secció destinada a explicar de forma clara els principals serveis de FoodLogistic S.A., amb una estructura orientada a empresa i clients potencials.

### Sobre nosaltres
Apartat pensat per reforçar la confiança, explicant de forma breu la identitat de l’empresa, el seu enfocament i els seus valors.

### Contacte
Secció amb formulari i informació bàsica de contacte per facilitar la comunicació amb clients o possibles col·laboradors.

### Informació legal
S’ha afegit una estructura bàsica orientativa amb continguts relacionats amb privacitat, legalitat i cookies, per tal d’apropar la proposta a un entorn web empresarial actual.

## Millores respecte a una web corporativa antiga

Aquesta proposta planteja una millora clara en diferents aspectes:

- **imatge visual més moderna**
- **millor organització dels continguts**
- **estructura responsive**, adaptable a ordinador, tauleta i mòbil
- **navegació més clara i directa**
- **presentació més corporativa i orientada a negoci**
- **possibilitat de mesurar visites i activitat** amb analítica integrada

## Tecnologies utilitzades

Per al desenvolupament d’aquest prototip s’han utilitzat eines web bàsiques i estàndard:

- HTML
- CSS
- Git i GitHub
- GitHub Pages per a la publicació
- StatCounter per al seguiment de visites

També s’ha utilitzat una eina d’assistència visual per generar una base inicial del disseny, posteriorment revisada i adaptada per ajustar-se als requisits de la pràctica i a la identitat corporativa proposada.

## Analítica i seguiment

S’ha integrat **StatCounter** com a sistema d’analítica amb comptador invisible. Aquesta eina permet disposar d’informació bàsica sobre:

- nombre de visites
- visitants únics
- pàgines vistes
- procedència de les visites
- dispositiu o navegador utilitzat

Aquest control és útil per entendre l’activitat inicial del lloc web i poder prendre decisions futures de millora.

## Valoració del prototip

El resultat d’aquesta proposta és una primera base funcional i visualment coherent per representar FoodLogistic S.A. a Internet d’una manera més actual, professional i alineada amb les necessitats d’una empresa del sector.

Es tracta d’un prototip pensat per mostrar l’aspecte, l’estructura i el potencial d’una futura web corporativa més completa.
