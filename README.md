# Web Corporativa de FoodLogistic S.A.

Aquesta és la proposta de web corporativa de **FoodLogistic S.A.**, una empresa del sector de la logística alimentària, creada dins del mòdul de *Projecte Intermodular*.

## Demostració

[Veure la web publicada](https://nitroxi.github.io/foodlogistic-web/)

## Tecnologies utilitzades

Per a la creació d’aquesta web s’han utilitzat les següents tecnologies i eines:

- TeleportHQ com a eina d’IA i generació visual inicial
- HTML5
- CSS3
- Git i GitHub
- GitHub Pages per al desplegament
- StatCounter per a l’analítica web

## Estructura de la web

La pàgina inclou els apartats següents:

- Inici
- Serveis
- Sobre nosaltres
- Contacte

El formulari de contacte inclou els camps següents:

- Nom
- Correu electrònic
- Telèfon de contacte
- Missatge

## Analítica

La web incorpora un comptador invisible de **StatCounter** per fer el seguiment de les visites i obtenir dades bàsiques d’ús.
